# Call Sheet Maker

Standalone browser app for building printable production call sheets.

## Editing Workflow

- Edit `index.html`, `styles.css`, `app.js`, and `logos.inline.js`.
- Run `npm run check` before and after meaningful changes.
- Run `npm run build` to regenerate the portable single-file `Call Sheet Maker.html`.

`Call Sheet Maker.html` is the release artifact. Avoid editing it directly because it is generated from the source files.
